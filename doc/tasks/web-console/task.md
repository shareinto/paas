# web-console 测试可用版本任务

## 目标

使用 React + TypeScript + Vite + Ant Design 实现 PaaS Web 控制台测试可用版本页面，所有底层系统能力只通过 PaaS API 暴露。

UI 风格必须参考根目录 `xx.png` 和 `doc/UI风格.md`，采用企业级运维控制台风格。
所有用户可见文案必须使用中文，不使用英文作为页面标题、导航菜单、按钮、表单标签、提示、状态、空状态或错误文案。

## 任务清单

- [x] 初始化 `web/console` 工程，使用 React、TypeScript、Vite。
- [x] 接入 Ant Design、React Router、TanStack Query、Zustand。
