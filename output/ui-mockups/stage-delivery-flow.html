绑定集群

绑定到租户级 Stage，立即进入该 Stage 的可选集群池，仅影响后续发布。同一集群可绑定多个 Stage。
保存后，该 Stage 下所有应用发布时都会从这些集群中选择目标子集。

添加 Stage

Stage key 创建后保持稳定，后续不允许修改；删除 Stage 会物理删除模板项和该 Stage 的集群池绑定，历史发布记录保留。

默认使用项目名称，用户可手动填写；保存为应用 Stage 集群配置。
创建后会为每个目标集群生成独立 Deployment 和 Argo CD Application 同步记录。
