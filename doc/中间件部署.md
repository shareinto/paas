# 中间件部署信息

## 开发环境服务器

```text
host = 192.168.171.131
os = Linux
container_runtime = Docker
docker_network = paas-net
base_dir = /root/paas-middleware
user = root
```

凭据文件：

```text
/root/paas-middleware/.env
```

说明：

- `.env` 文件保存在服务器本地，权限为 `600`。
- 不要把 `.env` 内容提交到代码仓库。
- 文档中不记录服务器登录密码、root 密码、MySQL 密码、Redis 密码等敏感信息。

## MySQL

```text
container_name = paas-mysql
image = m.daocloud.io/docker.io/library/mysql:8.0
host = 192.168.171.131
port = 3306
database = paas
user = paas
engine = InnoDB
charset = utf8mb4
collation = utf8mb4_unicode_ci
restart_policy = unless-stopped
data_dir = /root/paas-middleware/mysql
```

当前服务器 MySQL 用作开发阶段共享库。开发阶段 schema 大改、缺字段、缺表或迁移漂移时，允许按 `doc/MySQL迁移与启动流程.md` 清库重建；可使用 `./scripts/dev-up.sh --recreate-db` 让脚本删除并重建 `MYSQL_DATABASE` 后全量建表。重建会删除应用、构建、Freight、Stage、审计等业务数据。清库前必须通知相关使用者，并按需备份。

进入试点、生产或任何需要保留业务数据的环境后，不得使用清库重建处理 schema 变更，应通过新增 migration 升级。

连接信息从服务器凭据文件读取：

```text
MYSQL_HOST
MYSQL_PORT
MYSQL_DATABASE
MYSQL_USER
MYSQL_PASSWORD
MYSQL_ROOT_PASSWORD
```

验证命令：

```bash
docker exec paas-mysql mysqladmin ping -h 127.0.0.1 -uroot -p"$MYSQL_ROOT_PASSWORD"
docker exec paas-mysql mysql -uroot -p"$MYSQL_ROOT_PASSWORD" -e "SELECT VERSION(), @@character_set_server, @@collation_server;"
```

### 全量测试 MySQL

全量测试使用仓库脚本自动准备测试 MySQL：

```bash
scripts/test-full.sh
```

脚本行为：

- 如果已设置 `PAAS_TEST_MYSQL_DSN`，直接复用该 MySQL。
- 如果未设置 `PAAS_TEST_MYSQL_DSN`，自动启动 1 个临时 MySQL 容器。
- 所有 Go package 测试通过同一个 `PAAS_TEST_MYSQL_DSN` 复用该 MySQL，避免 `go test ./...` 为多个 package 同时拉起多个 testcontainers MySQL。
- Go package 默认按 `-p 1` 串行运行，避免共享 MySQL 下不同 package 的同名测试库互相影响；需要提速时可设置 `PAAS_GO_TEST_P`。
- 测试结束或中断时自动停止临时 MySQL 容器。
- 前端构建可能更新 `web/console/tsconfig.tsbuildinfo`；如果脚本启动时该文件是干净的，脚本退出时会自动恢复它。

可选环境变量：

```text
PAAS_TEST_MYSQL_DSN       # 已有测试 MySQL DSN；设置后脚本不会启动容器
PAAS_TEST_MYSQL_IMAGE     # 临时 MySQL 容器镜像，默认 m.daocloud.io/docker.io/library/mysql:8.0
PAAS_TEST_MYSQL_PASSWORD  # 临时 MySQL root 密码，默认 password
PAAS_GO_TEST_P            # go test package 并发数，默认 1
```

`PAAS_TEST_MYSQL_DSN` 示例：

```bash
export PAAS_TEST_MYSQL_DSN='root:password@tcp(127.0.0.1:3307)/mysql?charset=utf8mb4&collation=utf8mb4_unicode_ci&parseTime=true&loc=Local&interpolateParams=true'
```

## Redis

```text
container_name = paas-redis
image = m.daocloud.io/docker.io/library/redis:7.2-alpine
host = 192.168.171.131
port = 6379
appendonly = yes
restart_policy = unless-stopped
data_dir = /root/paas-middleware/redis
```

连接信息从服务器凭据文件读取：

```text
REDIS_HOST
REDIS_PORT
REDIS_PASSWORD
```

验证命令：

```bash
docker exec paas-redis redis-cli -a "$REDIS_PASSWORD" ping
```
